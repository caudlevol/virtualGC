The Replit Prompt to Expand the Configurator
Expand the supported configurator renovation types from 5 to 13. Add the following types to the configurator chip-selection flow, each with their own material/quality selection options:
Staircase: Options — Carpet treads / Hardwood refinish / New hardwood treads / Full replacement with new railings. Quality tiers — Basic / Mid / High-end.
Roof: Options — Patch repair / 3-tab asphalt replacement / Architectural shingle replacement / Metal roof. Size input — approximate square footage or use property sqft as default.
HVAC: Options — Repair / Single unit replacement / Full system replacement (furnace + AC) / Mini-split addition. Age input — current system age.
Deck / Patio: Options — Pressure-treated wood / Composite decking / Concrete patio / Stamped concrete. Size input — approximate square footage.
Garage: Options — Door replacement only / Epoxy floor / Full interior finish / Conversion to living space.
Basement Finishing: Options — Basic framing + drywall / Full finish with flooring + lighting / Full finish with bathroom add / Full finish with egress windows.
Exterior Paint / Siding: Options — Full exterior repaint / Trim only / Vinyl siding replacement / Fiber cement siding replacement.
Landscaping: Options — Basic cleanup + sod / Full front yard redesign / Full yard + hardscape / Driveway replacement.
For each new type, add the appropriate material cost rows to seed.ts using the same structure as the existing kitchen/bathroom entries. Use national average unit costs from RSMeans 2024 as the baseline. Apply the existing regional multiplier logic — no changes needed to the cost engine itself.
Also update the SYSTEM_PROMPT clarifying questions section to cover all new types with 1–2 targeted questions each, following the same pattern as the staircase/HVAC/roofing questions already added.
Bottom line: Going from 5 to 13 configurator types covers roughly 85% of what comes up in a real showing conversation. The remaining 15% (structural, full gut, custom additions) stays in the AI conversational path where it belongs. This is a meaningful product upgrade and it makes the "Locked Estimate" badge far more useful — agents can get a deterministic quote on almost anything a buyer asks about.
