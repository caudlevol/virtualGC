Gemini-24: Smart Scope Context-Aware Options and Dismissible Groups
In the Smart Scope configurator, we need to allow users to dismiss/remove specific option groups they don't want (e.g., if they only want a garage door but not the floor).
1. Update chat.tsx ConfiguratorChips component:
In artifacts/vgc-app/src/pages/chat.tsx, locate the ConfiguratorChips component (around line 495).
Add state for dismissed groups:
TSX
const [dismissedGroups, setDismissedGroups] = useState<Set<string>>(new Set());
Update the allSelected logic to ignore dismissed groups:
TSX
const activeGroups = config.groups.filter(g => !dismissedGroups.has(g.key));
const allSelected = activeGroups.length > 0 && activeGroups.every(g => selections[g.key]);
Update the handleSubmit logic to only send selections for active groups:
TSX
const handleSubmit = () => {
  if (!allSelected) return;
  
  // Filter out selections for dismissed groups
  const activeSelections = Object.fromEntries(
    Object.entries(selections).filter(([key]) => !dismissedGroups.has(key))
  );
  
  configuratorMutation.mutate({
    conversationId,
    data: { renovationType, selections: activeSelections },
  });
};
2. Add the Dismiss (Trash) button to the UI:
In the config.groups.map loop inside the return statement of ConfiguratorChips (around line 586), update the group header to include a dismiss button:
TSX
{config.groups.map(group => {
  if (dismissedGroups.has(group.key)) return null;
  
  return (
    <div key={group.key} className="space-y-2 relative group">
      <div className="flex items-center justify-between">
        <label className="text-xs font-semibold text-foreground uppercase tracking-wide">{group.label}</label>
        <button 
          onClick={() => {
            setDismissedGroups(prev => {
              const next = new Set(prev);
              next.add(group.key);
              return next;
            });
            // Also clear the selection if it was made
            setSelections(prev => {
              const next = { ...prev };
              delete next[group.key];
              return next;
            });
          }}
          className="text-muted-foreground hover:text-red-400 transition-colors p-1 rounded opacity-0 group-hover:opacity-100"
          title="Remove this option"
        >
          <Trash2 className="w-3.5 h-3.5" />
        </button>
      </div>
      <div className="grid grid-cols-2 sm:grid-cols-3 gap-1.5">
        {group.options.map(opt => {
          const isSelected = selections[group.key] === opt.label;
          return (
            <button
              key={opt.label}
              onClick={() => setSelections(prev => ({ ...prev, [group.key]: opt.label }))}
              className={\`px-3 py-2.5 sm:py-1.5 rounded-full text-xs font-medium transition-all border \${
                isSelected
                  ? "bg-primary text-white border-primary shadow-md shadow-primary/20"
                  : "bg-white/5 text-muted-foreground border-white/10 hover:border-primary/50 hover:text-foreground"
              }\`}
            >
              {opt.label} <span className="opacity-60 ml-1">{opt.price}</span>
            </button>
          );
        })}
      </div>
    </div>
  );
})}
Note: Make sure to import Trash2 and RotateCcw from lucide-react at the top of chat.tsx if they are not already there.
